# Ansible Role: Mutillidae 2

An Ansible Role that installs Mutillidae 2.x on Debian/Ubuntu.

## Dependencies

None

## Example Playbook
`
  - hosts: mutillidae
    roles:
      - { role: nickmk.mutillidae }
`
